🧠 v2 Engine — Strategies & Models

Antigravity v2 is a regime-aware gradient-boosted decision tree (GBDT) bot. It doesn't run hand-coded "strategies" like ORB or VWAP-pullback. Instead, every 5-minute bar is classified into a regime, and a specialist ML model trained specifically for that regime decides the direction.

How a bar gets a decision
  1. 5-min bar closes → NT8 pushes to Node
  2. 22 features computed (RSI, MACD, BB-z, ATR%, EMA distances, vol-Z, time-of-day, etc.)
  3. Regime classifier picks one: TREND_UP, TREND_DOWN, VOL_EXPANSION, or CHOP
  4. If CHOP → no trade. Otherwise → load the matching long & short specialist models
  5. Each specialist outputs a probability 0–1. If prob ≥ threshold → FIRE.
  6. Exit: 1.5×ATR stop, 2.7×ATR target (1.8:1 R:R), BE at 0.8×ATR profit, trail at 1.2×ATR.
